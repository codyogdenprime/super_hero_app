# super_hero_app
Code Challenge #5

## Install Guide

If you don't have [bower](http://bower.io).
```bash
npm install -g bower
```
After bower is installed...
```bash
bower install
```
...then...
```bash
npm install
```
...and to run it:
```bash
npm start
```

[http://localhost:4242/](http://localhost:4242/)
